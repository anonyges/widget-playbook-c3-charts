# Playbook C3 Charts

## Description

Output of the playbook is rendered to C3 charts.  
The return value(set variable) result must contain "data" as chart return value.  
For more available chart value check link: <https://c3js.org/examples.html>

### Render Page

- Dashboard
- View Panel
- Reports
